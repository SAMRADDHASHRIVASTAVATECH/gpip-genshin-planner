# Project Celestia

### Intelligent Genshin Impact Progression Planning Platform

An advanced progression-planning and account-management system focused on Adventure Rank optimization, character progression, exploration tracking, Primogem preparation, farming efficiency, and dynamic route generation.

</div>

---

# Features

## Progression Systems

- Adventure Rank progression planning
- Character acquisition planner
- Character/weapon/talent upgrade tracking
- Intelligent enemy farming routes
- Exploration progression management
- Region & Statue progression systems
- Primogem requirement estimation
- Dynamic progression execution order generation

---

# Exploration Systems

- Region exploration tracking
- Statue of The Seven progression
- Waypoint progression tracking
- Orb progression calculations
- Exploration percentage monitoring
- Route-based progression suggestions

---

# Account Systems

- Character roster management
- Party progression tracking
- Upgrade material planning
- Enemy EXP calculations
- Weapon progression systems
- Persistent progression storage

---

# Architecture

Project Celestia is built as a hybrid progression-management platform using:

- React
- TypeScript
- Vite
- Node.js
- Express
- SQLite

The system architecture, progression logic, planner integration, debugging, backend/frontend synchronization, deployment recovery, and runtime refinement were manually developed and expanded throughout the project lifecycle.

---

# Current Deployment

The currently deployed public version mainly acts as a showcase/demo build for the planner interface and progression systems.

The complete intended runtime experience — including:

- Local SQLite database systems
- Offline progression persistence
- Full synchronization systems
- Data import/update systems
- Local backend APIs

—is designed to run locally on the user's own machine.
